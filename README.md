## 📖 About

Ọjà-Wá is a production-ready e-commerce platform built to showcase full-stack development skills using Django REST Framework and React. It includes real-world features like JWT authentication, a Paystack payment integration, product search and filtering, order tracking, and a wishlist system — all wrapped in a clean, mobile-responsive Nigerian-themed design.

Built as a portfolio project to demonstrate backend API design, frontend state management, third-party payment integration, and full deployment pipelines.
